Basic Syntax

uname [OPTION]

Common Uses
Command	Meaning	Example Output
uname	Shows the kernel name only	Linux
uname -s	Same as above (kernel name)	Linux
uname -n	Shows the hostname (network node name)	myserver.local
uname -r	Shows the kernel release	6.4.0-42-generic
uname -v	Shows the kernel version	#43~22.04.1-Ubuntu SMP Wed Apr 3 11:11:11 UTC 2025
uname -m	Shows the machine hardware name (architecture)	x86_64
uname -p	Shows the processor type (sometimes same as -m)	x86_64
uname -i	Shows the hardware platform	x86_64
uname -o	Shows the operating system	GNU/Linux
uname -a	Shows all information in one line	Linux myserver.local 6.4.0-42-generic #43~22.04.1-Ubuntu SMP x86_64 GNU/Linux
Example

$ uname -a
Linux mahesh-laptop 5.15.0-89-generic #99-Ubuntu SMP x86_64 GNU/Linux

This tells you:

    Linux → Kernel name

    mahesh-laptop → Hostname

    5.15.0-89-generic → Kernel release

    #99-Ubuntu SMP → Build version

    x86_64 → Architecture (64-bit)

    GNU/Linux → OS
