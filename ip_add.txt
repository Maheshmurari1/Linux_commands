Common Usage
1. Show all network interfaces and their IP addresses

ip addr show
# or shorter
ip a

2. Show only IPv4 / IPv6 addresses

ip -4 addr show   # IPv4 only
ip -6 addr show   # IPv6 only

3. Show network interfaces (like ifconfig)

ip link show
# or
ip l

4. Bring an interface up or down

ip link set eth0 up
ip link set eth0 down

5. Assign an IP address to an interface

ip addr add 192.168.1.100/24 dev eth0

6. Remove an IP address

ip addr del 192.168.1.100/24 dev eth0

7. Show routing table

ip route show
# or shorter
ip r

8. Add a static route

ip route add 192.168.2.0/24 via 192.168.1.1

9. Delete a route

ip route del 192.168.2.0/24
